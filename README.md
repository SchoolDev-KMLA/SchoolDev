# SchoolDev

SchoolDev는 중고들학생들이 직접 만들고 이용하는 교내 인트라넷 앱입니다.  

이 페이지는 코딩에 약간의 관심만 있는 학생도 자기 학교의 인트라넷 앱을 쉽게 만들어볼 수 있도록 차근차근 돕는 것을 목표로 하며, 
배우고자 하는 의지가 강한 학생들에게는 더욱 구체적이고 심화된 내용까지 학습해볼 수 있는 기회를 제공하고자 합니다.
